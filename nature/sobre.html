Sobre o Projeto

Este site é apenas uma ideia de projeto que eu tive, para práticar alguns conceitos que estou aprendendo no desenvolvimento de site, usando HTML, CSS e JavaScript, principalmente envolvendo posicionamento de itens e exibição de imagens.

Todas as imagens usadas no projeto são do Pexels, um site com um grande banco de imagens e videos de alta qualidade, disponibilizado gratuitamente.
